Public document generator

Turn one prompt into a polished document.

This interface drives the backend planner, executor, and reflection loop to produce professional business documents and a downloadable .docx file. It is built to feel crisp, trustworthy, and easy to deploy for public use.

Compose a request

Give the agent a business task, and it will return a plan, assumptions, a reflection result, and a document download link.

Ready

The UI stays on the same page, calls the API asynchronously, and shows the plan and reflection results before linking the generated file.

Waiting for a request.
1. Plan

The backend turns the prompt into structured steps and assumptions, not free-form text.

2. Execute

Mock data is gathered, sections are drafted, and a clean Word document is produced.

3. Reflect

The draft is reviewed once and patched if there is a concrete gap worth fixing.

Live output

Run a request to see the generated plan, assumptions, reflection notes, and download link appear here.

Professional presentation

Responsive cards, calm spacing, and a restrained palette keep the interface credible for public use.

Direct downloads

Users get a single action result: a generated document they can preview, download, or hand off.

Deployable as-is

The frontend is served directly from FastAPI, so one app can power both the UI and the API.
